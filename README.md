## Installation

1. Clone the Repository





```bash
git clone https://github.com/Pawan-789/quizflow.git
cd quizflow

```
2. Install Dependencies


```bash

cd backend
npm install

```
3. Frontend Setup
```bash

cd ../frontend
npm install
```

4. Start the Application
(a). Run Backend
```bash
cd backend
node server.js
```
(b).Run Frontend
```bash 
cd frontend
npm run dev
```
